# Bootcamp Exercises: HTML, CSS, JavaScript, and React

This package contains 9 full practice projects:

- 3 HTML + CSS projects
- 3 HTML + JavaScript projects
- 3 React/Vite projects

Each folder includes runnable code and a README with goals, requirements, and stretch goals.

## Running HTML Projects
Open `index.html` directly in the browser.

## Running React Projects
From inside a React project folder:

```bash
npm install
npm run dev
```
