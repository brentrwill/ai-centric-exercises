# Restaurant Menu Website

Category: HTML + CSS  
Difficulty: Intermediate

## Goal
Create a visually organized restaurant menu page.

## Skills Practiced
- Tables vs cards
- Images/backgrounds
- CSS variables
- Section organization
- Responsive cards

## Student Tasks
1. Build menu sections for starters, entrees, desserts, and drinks.
2. Use cards to display menu items, descriptions, and prices.
3. Add a reservation form UI.
4. Use CSS variables for colors and spacing.

## Stretch Goals
- Add a featured item badge.
- Create a print-friendly CSS section.

## How to Run
Open `index.html` in a browser.
