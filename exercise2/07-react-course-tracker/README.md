# Course Tracker App

Category: React  
Difficulty: Beginner React

## Goal
Build a React app that tracks boot camp courses and completion progress.

## Skills Practiced
- Vite React setup
- Components
- useState
- Props
- Controlled forms
- List rendering

## Student Tasks
1. Create components for CourseForm, CourseList, CourseItem, and ProgressSummary.
2. Add new courses.
3. Toggle completed status.
4. Delete courses.
5. Show progress percentage.

## Stretch Goals
- Persist courses to localStorage with useEffect.
- Add difficulty and category fields.

## How to Run
```bash
npm install
npm run dev
```
