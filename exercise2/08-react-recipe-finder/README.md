# Recipe Finder App

Category: React  
Difficulty: Intermediate React

## Goal
Build a searchable recipe catalog using mock data.

## Skills Practiced
- Component composition
- useState
- Filtering data
- Conditional rendering
- Derived state

## Student Tasks
1. Render recipe cards from mock data.
2. Search by name or ingredient.
3. Filter by category.
4. Select a recipe to view details.
5. Show a useful empty state.

## Stretch Goals
- Add favorites.
- Add localStorage for favorites.

## How to Run
```bash
npm install
npm run dev
```
