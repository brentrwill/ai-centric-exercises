# Interactive To-Do List

Category: HTML + JavaScript  
Difficulty: Beginner/Intermediate

## Goal
Build a browser-based task list using DOM events and localStorage.

## Skills Practiced
- DOM selection
- Event listeners
- Array updates
- localStorage
- Form validation

## Student Tasks
1. Add tasks from an input field.
2. Mark tasks as complete.
3. Delete tasks.
4. Filter all, active, and completed tasks.
5. Persist tasks after refresh with localStorage.

## Stretch Goals
- Add edit task functionality.
- Add due dates or priority labels.

## How to Run
Open `index.html` in a browser.
