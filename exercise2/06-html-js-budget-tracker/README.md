# Budget Tracker

Category: HTML + JavaScript  
Difficulty: Intermediate

## Goal
Build a small expense/income tracker with calculations.

## Skills Practiced
- Forms
- Number parsing
- Array reduce
- Dynamic tables
- localStorage

## Student Tasks
1. Add income or expense transactions.
2. Display transaction list.
3. Calculate income total, expense total, and balance.
4. Delete transactions.
5. Save data to localStorage.

## Stretch Goals
- Add categories and filters.
- Add chart-like visual bars with CSS.

## How to Run
Open `index.html` in a browser.
