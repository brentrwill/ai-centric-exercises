# Responsive Profile Card

Category: HTML + CSS  
Difficulty: Beginner

## Goal
Create a polished personal profile card using semantic HTML and responsive CSS.

## Skills Practiced
- Semantic HTML structure
- CSS box model
- Flexbox layout
- Responsive design
- Hover states

## Student Tasks
1. Build a profile card with photo placeholder, name, title, bio, skills, and contact links.
2. Style the card with spacing, borders, shadows, and responsive layout.
3. Add hover effects to buttons and skills.
4. Make the design readable on mobile and desktop.

## Stretch Goals
- Add a dark mode version using a class on the body.
- Add a second card and place both in a responsive grid.

## How to Run
Open `index.html` in a browser.
