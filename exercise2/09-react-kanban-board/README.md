# Mini Kanban Board

Category: React  
Difficulty: Intermediate React

## Goal
Build a task board with To Do, In Progress, and Done columns.

## Skills Practiced
- State updates
- Reusable components
- Passing callbacks
- Filtering lists
- Project organization

## Student Tasks
1. Create Board, Column, TaskCard, and TaskForm components.
2. Add new tasks.
3. Move tasks between columns.
4. Delete tasks.
5. Show counts per column.

## Stretch Goals
- Add priority labels.
- Add drag-and-drop later as a separate challenge.

## How to Run
```bash
npm install
npm run dev
```
