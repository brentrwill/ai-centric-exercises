# JavaScript Quiz App

Category: HTML + JavaScript  
Difficulty: Intermediate

## Goal
Create a multiple-choice quiz with score tracking.

## Skills Practiced
- Arrays of objects
- Conditional logic
- Rendering from data
- Click events
- Score calculation

## Student Tasks
1. Display one question at a time.
2. Render answer buttons dynamically.
3. Track correct answers.
4. Show final score and restart option.
5. Disable double-answering.

## Stretch Goals
- Add a timer per question.
- Randomize question order.

## How to Run
Open `index.html` in a browser.
