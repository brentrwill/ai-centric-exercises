# Product Landing Page

Category: HTML + CSS  
Difficulty: Beginner/Intermediate

## Goal
Build a marketing landing page for a fictional product.

## Skills Practiced
- Page sections
- CSS grid
- Navigation layout
- Hero section
- Cards and CTA buttons

## Student Tasks
1. Create header, hero, features, pricing, testimonials, and footer sections.
2. Use CSS grid for feature and pricing cards.
3. Add buttons with hover states.
4. Make the nav and content responsive.

## Stretch Goals
- Add a FAQ section.
- Add a sticky header.

## How to Run
Open `index.html` in a browser.
